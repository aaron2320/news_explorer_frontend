# News_Explorer_Frontend

Application that allows users to search for news articles on many topics and save them. It was built with React.

## Overview

NewsExplorer was created to be a responsive web page that allows users to search for news articles. They can create a personal account and save their favorite articles.

Getting Started

1. Clone the repository:

```bash
git clone https://github.com/your-username/news-explorer-frontend.git
```

2. Install dependencies:

```bash
cd news-explorer-frontend
npm install
```

3. Start the development server:

```bash
npm start
```

## Usage Guide

1. **Search for Articles:**

   - Enter a topic in the search bar
   - Click the "Search" button
   - Browse through the results

2. **Create an Account:**

   - Click "Sign in" in the navigation bar
   - Choose "Sign up" option
   - Fill in your details

3. **Save Articles:**

   - Sign in to your account
   - Search for articles
   - Click the bookmark icon on any article to save it

4. **View Saved Articles:**
   - Navigate to "Saved articles" in the header
   - View all your saved articles
   - Remove articles using the trash icon
